faqs:
  - question: "What is a dock dashboard?"
    answer: "A dock dashboard is a real-time scheduling view that shows every inbound and outbound appointment at a warehouse by dock door, carrier, and time. It replaces spreadsheets and whiteboards with a live interface the whole operations team can see and update. DataDocks' dock dashboard combines calendar views, color-coded appointments, and drag-and-drop rescheduling in one place."
  - question: "What calendar views does the DataDocks dock dashboard have?"
    answer: "Seven: yard view, dock view, day, week, month, overhead property view, and capacity-per-timeslot. Each view shows the same appointment data from a different angle, and every view can be customized with the specific fields and information your team needs to see. Most facilities use day and dock views for live operations and the week, month, or capacity view for planning ahead."
  - question: "Can multiple users view and edit the dock dashboard at the same time?"
    answer: "Yes. The dashboard updates live for every user — a dock coordinator moving an appointment to a new door, a gate guard checking in a truck, and a manager reviewing the week all see the same state in real time. No refreshes, no stale spreadsheets sitting on someone's desktop."
  - question: "Can the dock dashboard be personalized by role?"
    answer: "Yes. Each user can set the default view, filters, and columns they see. A dock coordinator can focus on today's live appointments, a planner can see the whole week across doors, and a carrier-side user can see only their own loads. Preferences persist per user so the dashboard opens to the right view every time."
  - question: "How does drag-and-drop rescheduling work?"
    answer: "Appointments can be dragged to a different door, time slot, or day directly on the dashboard. If a move would break a scheduling rule — capacity limits, door restrictions, carrier-specific requirements — DataDocks warns your team before saving so any override is deliberate, not accidental. Carriers and external customers booking through the portal can't override rules; only your internal team can choose to overbook."
  - question: "What do the colors on the dock dashboard mean?"
    answer: "Colors map to appointment status and load type so you can read the state of the entire dock at a glance. This lets managers spot active doors, waiting trucks, and early bottlenecks without opening individual appointments one by one."
